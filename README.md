# common-mark

CommonMark WebComponents.

Use: https://github.com/commonmark/commonmark.js

```html
<common-mark>
# test

CommonMark sample.
</common-mark>
```

```html
<common-mark src="./sample.md"></common-mark>
```

## Develop

### Need

* `tsc`
  * TypeScript
* `npm`
  * Node.js
* `uglify-js`

### Build

```sh
deno task build
```
